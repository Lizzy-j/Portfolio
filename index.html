Developer + DevOps Lead, Criminal Appeals Fund

As part of the Tech for Better programme organised by Founders and Coders, I worked in a team of 4 to develop a single-page web application for the Criminal Appeals Fund. Developer + UX Lead, Where's Whippy

A single-page web application connecting local ice cream vendors with customers using the Google Maps API. The project is built with React and Styled Components in the front-end. Whilst the back-end was built with PostgreSQL Express.js. I was responsible for the implementation of the Google Maps API to render geolocation data of customers for vendors. As the UX lead, I led user testing and ensured the app met the accessibility guidelines outlined by W3C.
